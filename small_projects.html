Paw Bubbles


Project information

  • Category: Top Down
  • Engine: Unreal Engine 5
  • Project date: 2025
  • Itchio: Link
  • GameJam

About the Project

You play as a little cat defending her island from invading monsters using bubble magic. She traps enemies in bubbles, then dashes into them to set off chain reactions—longer chains mean higher scores.

Unreal Engine is not a good engine for making a jam game, especially if, like me, you prefer programming in C++ rather than simply using Blueprints. The compilation times slowed the project down and prevented me from achieving the level of polish I would have liked. Despite everything, the result is an interesting game: the mechanic of chaining bubble explosions is fun, and the twin-stick movement feels satisfying. Even with its shortcomings, I'm glad I carried out this small project, almost entirely on my own as far as programming is concerned.

ScrapHell


Project information

  • Category: Bullet Hell
  • Engine: Unity
  • Project date: 2024
  • Academic Project

About the Project

ScrapHell puts the player in the skin of a girl fighting against the police forces trying to kill her, the player controls the car in an attempt to survive against the ferocious forces by driven, dodging and of course eliminating the enemies.

ScrapHell features a high level of destructibility which, together with visual effects, creates a strong sense of spectacle. Movement is always constant along an infinite road, which greatly simplified development and allowed me to focus on the vehicle's lateral driving with drifting, as well as on enemy AI. The enemies use rubber banding so they never fall behind the player, adding tension to the driving and destruction. Finally, I'd like to highlight the use of post-processing effects such as chromatic aberration and a subtle noise filter to achieve the desired aesthetic.

Squid Circuit


Project information

  • Category: Weird Controls
  • Engine: Unity
  • Project date: 2024
  • Academic Project

About the Project

This project had only one rule: the controls had to be “weird.” From that came Squid Circuit, a game where two players share the same controller—one trying to escape a deadly circuit, while the other uses the environment's traps to kill the poor squid.

In this game, Player 1 navigates through a dynamic circuit, dodging bullets and avoiding obstacles to progress and reach the end within a set time, while Player 2 uses these elements to obstruct Player 1's progress. Upon dying, Player 1 respawns at the beginning of the circuit, allowing continuous attempts without restarting the entire level. The camera smoothly follows Player 1, ensuring visibility and constant control, creating a gaming experience that combines skill, strategy, and adaptability.

Rainbow Labyrinth


Project information

  • Category: Puzzle
  • Engine: Unreal Engine 5
  • Project date: 2023
  • Academic Project

About the Project

This school project focuses on the creation of Rainbow Labyrinth, a puzzle video game designed to challenge players with color-based riddles. The game features a dynamic maze where players must solve puzzles to reach the exit.

As my first completed project in Unreal Engine, Rainbow Labyrinth served as a foundation for everything that came after. While it may not feature large-scale implementations, it was a pivotal moment to delve into the basic fundamentals and learn to make the most of them. Elements like Unreal's Timeline and AnimMontages were used to immerse the player in the experience, while other tools streamlined the development process for me.

The Opposite of Victory


Project information

  • Category: Infinite Runner
  • Engine: Unity
  • Project date: 2023
  • Academic Project

About the Project

This school project focuses on the creation of an Infinite Runner. By definition, this genre has no win state, which is a concept not often explored in video games. I wanted to explore this idea by making the game an endless, seamless loop with no breaks—a perpetual state of failure, or in this case, whatever the opposite of victory is.

Technically, the most challenging part of this project is resetting the loop after each death without loading a new scene. Beyond that, the terrain generation is made up of a limited number of pieces that are randomly recycled, and visual effects are used to convey a sense of high speed, especially at higher difficulty levels.
